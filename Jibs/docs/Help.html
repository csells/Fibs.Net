How To Get Help

Support Requests and Questions

Check the Wiki and the FAQ. If your question isn't answered there, subscribe to the Java User mailing list and send an email.

Bugs

Check the Java or issue tracker for an existing ticket. If there is an existing issue, feel free to add a comment. If there is no existing issue for the bug,create a account and add a new ticket.

Feature Requests and Improvements

Check the Java  issue tracker for an existing request for a new feature or improvement. If there is an existing request, feel free to add a comment or vote for the request. If there is no existing request, you can to subscribe to the Developers mailing list and send an email describing the feature request.
